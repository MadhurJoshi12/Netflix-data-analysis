# Netflix data analysis
This project analyzes Netflix content data and applies unsupervised learning to uncover hidden patterns using KMeans. After cleaning and preprocessing the dataset, numerical features were scaled using StandardScaler to ensure effective clustering.  The model groups similar content into clusters based on underlying patterns, without relying on labeled data. The optimal number of clusters is determined using the Elbow Method, and the results are visualized to better understand the distribution of different groups.  This project demonstrates the practical application of clustering techniques to real-world data, highlighting how machine learning can be used to discover insights and structure within large datasets.
