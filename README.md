# 🎬 Netflix Data Clustering Analysis (Unsupervised Learning)

This project explores Netflix content data and applies Machine Learning techniques to uncover hidden patterns using clustering.

---

## 📌 Overview

How can we group similar Netflix content without predefined labels?

In this project, I used unsupervised learning to analyze and segment Netflix data into meaningful clusters based on underlying patterns.

---

## ⚙️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

---

## 🧠 Approach

- Loaded and explored the dataset  
- Selected numerical features for analysis  
- Handled missing values to ensure clean input  
- Scaled features using StandardScaler  
- Applied K-Means clustering algorithm  
- Used Elbow Method to determine optimal number of clusters  
- Visualized clusters for better understanding  

---

## 📊 Visualizations

- 📉 Elbow Method graph (WCSS vs K)  
- 📍 Cluster scatter plots  
- (Optional) PCA-based cluster visualization  

---

## 📈 Results

- Successfully grouped Netflix content into distinct clusters  
- Identified patterns and similarities within data  
- Demonstrated how unsupervised learning reveals hidden structure  

---

## 🧠 Key Learnings

- Importance of feature scaling in clustering  
- Understanding how K-Means forms clusters  
- Using Elbow Method to select optimal K  
- Visualizing clusters improves interpretability
